📁 Como usar imagens premium

Crie uma pasta chamada figurinhas no mesmo local deste HTML.
Salve as artes premium neste padrão:

figurinhas/BRA_01.webp
figurinhas/BRA_02.webp
figurinhas/ARG_01.webp

Também funciona com PNG/JPG se você alterar a extensão no código. Por padrão está em .webp.

Para usar como app no tablet, rode python start_tablet_server.py no computador, abra o endereço mostrado no tablet e use a opção de instalar/adicionar à tela inicial.

O HTML carrega a imagem premium mesmo quando a figurinha ainda está faltando; nesse caso ela aparece esmaecida para ajudar na conferência do álbum físico.
